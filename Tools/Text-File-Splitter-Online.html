❓ Frequently asked questions about file splitting

Can I split a CSV file without losing header?

Yes — the splitter treats files as raw text; you can manually keep headers by splitting after line 1. We suggest using “by lines” mode.

Is it really free? Any file size limit?

Always free, no signup. Browsers support files up to 500 GB (depends on free memory).

What formats are supported?

Any text‑based file: .txt, .csv, .log, .json, .xml, .md, .sql — even source code.

How is zero upload possible?

All processing uses the File API and Web Workers (if needed). File data never leaves your device.
